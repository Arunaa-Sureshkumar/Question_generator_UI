The Question Generator

How to Generate New Question

With this new question can be created with respect to ranges and multiples provided
  • Firstly , Enter the question with varying variables representing with keyword called var
  • Secondly , Enter Formulas to solve the above question
  • Thirdly , You will be having options which denote the 4 MCQ options among the answers, where formulas can be provided to make valid options
  • Fourth denotes variable which denotes how many variables in the question
  • Maximum or array and minimum denotes ranges of values where you provide maximum and minimum value , sometimes for fraction question sets of values can be used to variable in the question , in those cases maximum can be used by leaving and minimum value can be leave empty
  • Then, click enter , the download popup will be opened and you can download the excel sheets

The Paraphrase Generator

How To Generate Paraphrase of the Question

It paraphrases the question you provided below, the maximum number of questions can be paraphrased is 50.
  • From dropdown , select paraphrase option
  • Then , enter the question in the input box in the left side
  • Click send button after you entered the question
  • Wait until the response received , Voila you will get the response in the right side and you can download the csv file.
